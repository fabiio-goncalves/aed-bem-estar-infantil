# AED - Bem-Estar Infantil

## 📊 Análise do Impacto da COVID-19 no Bem-Estar Psicológico Infantil

Este projeto tem como objetivo analisar o impacto da pandemia de COVID-19 no bem-estar psicológico de crianças em idade pré-escolar e do 1.º ciclo.

A análise foi realizada com base em dados recolhidos através do questionário **AED_CP19_Saúde**, explorando variáveis sociodemográficas e indicadores emocionais e comportamentais.

---

## 🎯 Objetivos

* Analisar variáveis como género, idade e ciclo escolar
* Estudar padrões de sono das crianças
* Avaliar comportamentos emocionais (irritabilidade, obediência, etc.)
* Identificar relações entre variáveis relevantes

---

## 🧪 Metodologia

O projeto foi desenvolvido em várias etapas:

### 📌 Tratamento de dados (Excel)

* Correção de valores inválidos
* Substituição de valores omissos
* Criação de tabelas de frequências
* Tabelas dinâmicas (pivot tables)

### 📊 Análise estatística (Jamovi + R)

* Análises univariadas
* Análises bivariadas
* Cálculo de estatísticas descritivas (média, mediana, desvio padrão, etc.)
* Construção de gráficos (barras e distribuições)
